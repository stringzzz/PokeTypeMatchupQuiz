This quiz game is for learning the type effectiveness chart in Pokemon.

It randomly selects the attacking type and 1 or 2 defending types.
Choose the correct effectiveness and submit your answer.
It will tell you if you were correct or not, and if not it shows the correct answer.
Keeps track of how many you get correct/wrong.

A future version might show the effectiveness of the type against each defending type individually when a wrong answer is given.
Also could keep track of a list of all wrong answers and have the option to output that list.
